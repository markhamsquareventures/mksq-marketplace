# Markham Square Marketplace

Curated Claude Code plugins for skills, workflows, and productivity tools created by the Markham Square team.

## Installation

Add this marketplace to Claude Code:

```bash
/plugin marketplace add markhamsquareventures/mksq-marketplace
```

## Available Plugins

### Essentials

**Description:** Essential skills, commands, and hooks tailored to working in Laravel projects

**Categories:** Laravel, Inertia.js, React.js, Collaboration, Meta

**Install:**

```bash
/plugin install essentials@mksq-marketplace
```

**Repository:** https://github.com/markhamsquareventures/essentials
